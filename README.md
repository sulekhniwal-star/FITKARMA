## ⚡ Tech Stack

- **Framework**: Flutter 3.x
- **State Management**: Riverpod 2.x
- **Database**: Drift (SQLite) with AES-256 Encryption
- **Backend**: Appwrite (CLI-first)
- **AI**: Anthropic Claude (via server-side Functions)
- **Monetisation**: RevenueCat
